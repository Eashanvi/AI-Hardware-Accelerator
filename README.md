# AI Hardware Accelerator (FPGA)

## Overview

A Verilog-based AI accelerator implementing parallel neuron computation using MAC units.

## Features

* Multiply-Accumulate (MAC)
* Parallel neuron architecture
* FSM-based control
* Weight ROM
* Pipelined design

## Tools Used

* Xilinx Vivado
* Verilog HDL

## Output Example

Inputs: x1=1, x2=2, x3=3
Outputs: out1=6, out2=12, out3=18
